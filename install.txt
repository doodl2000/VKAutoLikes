1. Установка расширения User JavaScript and CSS на браузер (https://chromewebstore.google.com/detail/user-javascript-and-css/nbhcbdghjpllgmfilhnhkllmkecfmpld?hl=ru&utm_source=ext_sidebar)
2. В расширении добавить новое правило
3. Добавить url паттерн (https://vk.com/*)
4. Добавить в расширении модуль (https://cdn.jsdelivr.net/npm/sweetalert2@11.14.5/dist/sweetalert2.min.js)
5. Скопировать код из ./script.js и вставить в поле для JavaScript кода
6. Скопировать код из ./SweetAlert2.css и вставить в поле для CSS кода
7. Сохранить правило
